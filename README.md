# 📊 Portfólio de Dashboards Empresariais

Uma coleção abrangente de **dashboards profissionais de análise de negócios** construídos com tecnologias modernas de visualização de dados.  
Este portfólio demonstra expertise na criação de **painéis interativos em tempo real** em múltiplos domínios organizacionais, incluindo **Service Desk, Recursos Humanos, Finanças, Operações em Saúde, Gestão de Projetos e Melhoria Contínua de Processos**.

---

# 🎯 Propósito

Este repositório apresenta um conjunto de **dashboards analíticos voltados para tomada de decisão**, projetados para fornecer **insights acionáveis** em diferentes funções de negócio.

Principais objetivos:

- 📈 Visualização de dados em tempo real para suporte à decisão
- 🧠 Interfaces intuitivas para stakeholders técnicos e não técnicos
- ⚙️ Arquitetura escalável para múltiplas fontes de dados
- 🚀 Alta performance para análise de grandes volumes de informação
- 🎨 Estética profissional alinhada aos padrões corporativos

---

# 🛠 Tecnologias Utilizadas

| Tecnologia | Propósito |
|-------------|-------------|
| 🐍 **Python** | Lógica backend e processamento de dados |
| 🐼 **Pandas** | Transformação, limpeza e agregação de dados |
| 📊 **Plotly** | Visualizações interativas avançadas |
| ⚡ **Dash** | Aplicações web analíticas e dashboards corporativos |

# 📊 Portfólio de Dashboards

## 🛠 Service Desk Dashboard

Dashboard voltado para **monitoramento de operações de suporte de TI (ITSM)**.

Principais indicadores:

- Volume de chamados
- SLA de atendimento
- MTTR (Tempo Médio de Resolução)
- Distribuição por prioridade
- Tendência de incidentes

Tecnologias utilizadas:

`Python • Dash • Plotly • Pandas`

---

## 👥 RH – Benefícios

Indicadores:

- Utilização de benefícios

- Indicadores de RH

Tecnologias utilizadas:

`Python • Streamlit • Pandas`
